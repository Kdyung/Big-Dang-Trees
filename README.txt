BIG BUTT TREES
An entry for the Fall 2013 UCSD VGDC Game Jam. Created in FlashDevelop using the Flixel library.

Description:
You are a hat trying to get back to its owner. Navigate the world of steam and watch out for bats!

Controls
W,A,S,D - Movement
